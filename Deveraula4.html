O que é HTML?

HTML é uma das linguagens utilizadas para o desenvolvimento de websites. A origem da palavra vem do inglês,e em português traduz-se: Linguagem de Marcação de Hipertexto.
O HTML é a liguagem base da internet e seu objetivo é facilitar a comunicação entre os seres humanos e as máquinas. Sistemas como o Google podem exemplificar sua função, devido a gama de atividades que ele realiza.

O que é CSS?

O CSS é uma linguagem de folhas e estilos, que tem a função mais específica de trabalhar a formatação e layout.
A criação do CSS se deu basicamente por uma questão de organização das atribuições do HTML, que trabalha mais a questão da estruturação do código, enquanto que o CSS é voltado para o aperfeiçoamento da formatação. Os dois se integram e são utilizados de forma harmoniosa.
